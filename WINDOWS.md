# react-native-pdf-view

### Installation

* In mainpage.cs

```using ReactNative.Modules.PDFView;

return new List<IReactPackage>
                {
                ....
                new PDFViewPackage()
                };
                
```
                
Add 

PDFView.cs, 
PDF_Render.xaml, 
PDF_Render.xaml.cs 

to your project
